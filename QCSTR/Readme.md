# Experiment Overview
```
Algorithm: SART, FBP, QTR, QCSTR
Test sample: 30 by 30 Shepp-Logan phantom
Projection type: Radon transform
Quantum optimization model: Energy difference between a superimposed sinogram and an expemental sinogram
```
