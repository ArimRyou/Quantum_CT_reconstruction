# Experiment Overview
```
Test sample: 30 by 30 Shepp-Logan phantom
Projection type: Radon transform
Quantum optimization model: Energy difference between a superimposed sinogram and an expemental sinogram
```

## File Description
```
1. QCT_30by30_18prj - QCT_30by30_30prj: 
   CT image reconstructed by quantum algorithm according to the number of projections
2. QCT_30by30_10qbt:
   CT image reconstructed by quantum algorithm when each pixel of CT image is superimposed with 10 qubits
```
